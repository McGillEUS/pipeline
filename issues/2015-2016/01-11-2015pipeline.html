EUS Photography Committee

The EUS is interested in creating a photography committee dedicated to documenting all of the awesome events in engineering! Part of this task includes having skilled photographers ready to be let loose at any event. As such, we would like to host a photography tutorial dedicated to training these future superstar photographers. 

If you would like to be one of these glorious photographers, please fill out this form. We need to gauge interest in the committee before we can move forward with planning a tutorial.

Please note that it is highly preferred that you have a digital single-lens reflex camera [DSLR] with external flash for this tutorial, so you can truly gain hands-on experience working with the camera. If you are still interested but do not have a camera, please fill out the form anyways, and we will see what we can do!
