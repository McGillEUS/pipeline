Computational image analysis of digital nanoarray of single exosomes

April 1
Academic
Job Posting

We are looking for a research assistant for the summer with a strong programming background to develop and streamline an image analysis pipeline for high throughput fluorescence microscopy images of nanoarrays of extracellular vesicles and of proteins with single molecule sensitivity. The goal of the project is to improve and streamline the image analysis pipeline in conjunction with improvement of the experimental setup is optimized. Proficiency in Matlab is required as the existing code is in Matlab. While a basic framework of the analysis has been tested, the process would benefit from streamlining, and new features such as image registration and focus detection need to be implemented for quantitative, accurate and reproducible results. Start and end dates are negotiable but a minimum 3 months commitment is required. Email philippe.decorwin-martin@mail.mcgill.ca if interested.
